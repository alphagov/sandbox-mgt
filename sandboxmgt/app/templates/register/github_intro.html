GitHub account?

You log-in to your Sandbox using your GitHub account because:

  • Code that you write in your sandbox should be saved to GitHub, so you'll need an account for that anyway
  • Reusing a GitHub account saves you remembering yet another password
  • GitHub has excellent security, including 2 factor authentication

On the next screen (currently branded "Auth0"), please click "log in with GitHub".
